WB Anti‑Fake Bot

Телеграм‑бот принимает ссылку/артикул Wildberries и за ~3 секунды показывает:
- **Trust Score** (эвристики накрутки/подозрительности)
- **Выжимку отзывов**: топ‑плюсы / топ‑минусы

---

Как пользоваться

Отправьте боту:
- ссылку вида `https://www.wildberries.ru/...`
- или просто артикул.
